Electroencephalography (EEG) Data Classification

This repository details a process for classifying EEG data using Principal Component Analysis. It includes loading and preprocessing EEG data, followed by dimensionality reduction using PCA. The notebook also explores two machine learning models, Gradient Boosting Machine (GBM) and Random Forest, to classify the reduced EEG data, highlighting their moderate accuracy and class bias tendencies. Additionally, a simple Convolutional Neural Network (CNN) model is implemented and trained, showing an improving trend in accuracy but potential overfitting issues, as indicated by the performance disparity between training and validation datasets.
